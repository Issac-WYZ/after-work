# Art Pipeline

本文件记录《下班以后》的美术资源原则和后续正式美术规范入口。当前阶段只建立规则，不制作正式素材。

## 阶段原则

### 0.1-0.3

- 只使用 placeholder 占位素材。
- 不追求正式美术。
- 占位素材统一放在 `assets/placeholder/`。
- 不引入版权不清楚的素材。

### 0.4

开始制定正式美术规范，包括：

- 像素规格
- 角色比例
- Tile 尺寸
- UI 风格
- 命名规范
- 资源导入规范

### 0.5+

逐步替换正式素材，优先顺序为：

1. 主角
2. 出租屋
3. UI
4. 公司
5. 便利店
6. NPC
7. 城市地图

## 目录规则

- `assets/placeholder/`：占位素材。
- `assets/art/`：正式美术素材。
- `assets/audio/`：音效和音乐。
- 占位素材和正式素材不能混放。

## 版权规则

- 第三方素材必须记录来源、作者、许可证和链接。
- 版权不明确的素材不得进入仓库。
- 素材许可证记录在 `docs/asset_license_log.md`。
- 音频许可证记录在 `assets/audio/licenses.md`。

## 格式规则

- 图片优先使用 `.png`。
- 短音效优先使用 `.wav`。
- 音乐优先使用 `.ogg`。
- 字体优先使用 `.ttf` 或 `.otf`。
