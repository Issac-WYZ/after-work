# Roadmap

本路线图用于控制项目长期方向。任何阶段都应先保证当前版本闭环可运行，再考虑扩展内容。

## 0.1 最小可玩日常闭环

- 三个基础场景：Apartment、Office、ConvenienceStore。
- 玩家基础移动、碰撞和摄像机跟随。
- 通过交互点切换场景。
- 全局状态：Day、Time、Money、Energy、Mood、Stress。
- 工作小游戏：完成 5 次正确方向键输入后获得工资。
- 便利店购买：便当、咖啡、甜品。
- 睡觉进入第二天。
- HUD 实时显示状态。

## 0.2 日常反馈扩展

- 增加更多商品和简单状态反馈。
- 增加更清晰的交互提示。
- 调整数值平衡。
- 增加基础音效和更完整的 placeholder 美术。

## 0.3 房间生活扩展

- 增加出租屋内更多日常交互。
- 增加简单家具功能，但不做完整装修系统。
- 增加更多时间段反馈。

## 0.4 工作与消费扩展

- 增加工作小游戏变化。
- 增加更多消费选择。
- 引入更明确的收入、压力、心情取舍。

## 0.5 有限城市地点扩展

- 增加更多固定地点。
- 使用地点选择或小型连接场景，不做开放世界。
- 增加地点开放条件。

## 0.6 社交雏形

- 增加少量固定 NPC。
- 增加基础好感度和简单对话。
- 不做复杂剧情线。

## 0.7 成长与生活目标

- 增加学习、兼职或简单能力成长。
- 增加阶段性目标。
- 增加更多生活选择带来的状态变化。

## 0.8 居住与职业扩展

- 增加搬家、跳槽或自由职业雏形。
- 增加更长期的金钱规划。
- 增加更多可重复日常内容。

## 0.9 系统整合与内容补齐

- 整合日常、工作、消费、关系和成长系统。
- 补齐 UI、音频、美术、导出配置。
- 做完整可玩性测试。

## 1.0 原型完整版

- 形成稳定的现代生活模拟核心体验。
- 支持 macOS 和 Windows 版本导出。
- 具备较完整的日常循环、成长目标和可扩展内容结构。
